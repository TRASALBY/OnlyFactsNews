desc<테이블 명> : 테이블에 어떤 필드들이 정의되어 있는지 확인
list : 바로 직전에 실행시켰던 명령을 출력
run: 바로 직전에 실행했던 명령을 다시 실행
save <파일명> : 명령문을 파일로 저장
get <파일명> : 저장된 명령문을 불러옴
host : 운영체제 명령을 실행, exit로 복귀


create tablespace <테이블스페이스 명> datafile '<경로명>' size <크기>  : 경로에 크기를 가진 테이블 스페이스 생성
alter tablespace <테이블스페이스 명> add datafile '<경로명>' size <크기> : 테이블스페이스에 새로운 데이터파일 생성
drop tablespace <테이블스페이스 명> : 테이블스페이스 삭제
select <테이블 스페이스 명> : 조회

create user <계정 명> identified by <비밀번호> default tablespace <사용할 테이블스페이스> quota <용량> on <사용할 테이블스페이스>
: 계정을 생성하고 테이블 스페이스의 용량을 할당한다.
grant connect, resource to <계정 명>: 권한 부여

SQL : Structured Query Language
DDL : Data Definintion Language - 데이터 정의 언어, 저장 구조 명시 ex)스키마 정의, 수정, 삭제
DML : Data Manipulation Language - 데이터 조작 언어, 데이터 접근 조작 ex)레코드 검색, 삽입, 수정, 삭제
